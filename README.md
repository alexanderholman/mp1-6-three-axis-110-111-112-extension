# MP1.6 Three-Axis Extension (110|111|112)

Three-axis extension of MP1 with poly-axis-capable geometries in silicon.

## Repository role

- GitHub repo: `alexanderholman/mp1-6-three-axis-110-111-112-extension`
- Program: MP1 poly-axis interface analysis in Si
- Scope: Micro-paper execution unit.

## Getting started

- Read `plan.md` (micro) or `plan_micro_papers.md` (macro).
- Review `tasks/wip.md` and `tasks/triage.md` for current execution state.
- Follow setup notes in `INSTALL.md`.
